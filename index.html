Hamming SEC-DED Code Simulator

How It Works

  1. Select Data Length: Choose 8, 16 or 32 bits to define the input size.
  2. Enter Binary Data: Provide your binary message (e.g. 10110011). The length must match your selection.
  3. Generate Hamming Code: Computes SEC-DED parity bits and displays the encoded code.
  4. Flip Bit: Specify a position (1=rightmost) to introduce a single-bit error.
  5. Inject & Correct: Applies the error, recalculates syndrome, identifies faulty bit, and corrects it.
  6. Visualize Bits: Renders a two-row grid showing bit numbers and values. The error bit is highlighted in red and the corrected position in green.
